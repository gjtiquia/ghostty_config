# Ghostty Config

## where i belong

`~/.config/ghostty/`
 
## setup

`~/.config/ghostty/` should exist if you have opened ghostty before

```
# go into the existing config folder, should exist if you have opened ghostty before
cd ~/.config/ghostty`

# remove existing contents
rm config

# clone into this directory
git clone <repo-url> .
```

reload config with `ctrl+shift+,` (Linux) or `cmd+shift+,` (macOS)
